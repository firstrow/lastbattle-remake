# Last Battle Remake for Linux

This is my remake of Last Battle spectum game. 1997 year.

## Usage:
``` sh
make run
```

## Keybindings
```
F12 - generage new world
F11 - toggle fog of war
F2 - save game

For anything else - read source code.
```
